# HatsuneMikuBot

Discord Bot based off of Hatsune Miku, written in python!

![Miku wink](miku-wink.gif)

## Setup Instructions

- Install dependencies:

  pip install -r requirements.txt

- Create .env file:

  - Insert `token=<YOUR TOKEN HERE>`

- Start the bot:
  - Go to the terminal and type `python bot.py`
